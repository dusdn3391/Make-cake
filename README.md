# 🎂 케이크 굽기 게임

달콤한 나만의 케이크를 만들어보는 미니 웹 게임

---

## 🕹️ 게임 방법

1. **재료 넣기** — 올바른 재료 6가지를 볼에 넣어요
2. **반죽 붓기** — 버튼을 꾹 눌러 반죽을 80%만큼 부어요
3. **오븐 굽기** — 목표 시간에 맞춰 케이크를 꺼내요
4. **결과 확인** — 얼마나 잘 만들었는지 확인해요!

---

## 🏆 점수 기준

| 단계 | 성공 조건 |
|------|----------|
| 재료 넣기 | 올바른 재료 6가지만 선택 |
| 반죽 붓기 | 80~85% 사이로 붓기 |
| 오븐 굽기 | 목표 시간 ±0.5초 이내에 꺼내기 |

- ✅ 3개 성공 → `perfect.png`
- ✅ 2개 성공 → `normal.png`  
- ✅ 1개 이하 → `fail.png`
